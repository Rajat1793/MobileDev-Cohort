# Sensify — Release Notes

Paste the relevant version into Play Console → Release → "Release notes"
(max 500 characters per language). The `<en-US>` tag format is what the
Console expects.

---

## v1.0.0 (versionCode 2) — Initial release

### Play Console format
```
<en-US>
Welcome to Sensify! 🎉

Turn your phone's hidden sensors into five fun mini-games:
• Tilt Ball — roll a ball by tilting (accelerometer)
• Gyro Balance — steady the beam (gyroscope)
• Light Control — play with the light sensor
• Compass Hunt — match the heading (magnetometer)
• Shake Detector — build your shake streak

100% offline. No ads. No sign-up. No tracking.
</en-US>
```

### Short version (if you prefer something tighter)
```
<en-US>
First release of Sensify! Five mini-games powered by your phone's sensors —
tilt, gyroscope, light, compass and shake. Fully offline, no ads, no sign-up.
</en-US>
```

### Plain text (for changelogs / GitHub releases)
First public release of Sensify. Five sensor-powered mini-games: Tilt Ball
(accelerometer), Gyro Balance (gyroscope), Light Control (light sensor),
Compass Hunt (magnetometer), and Shake Detector (device motion). Fully
offline with no ads, accounts, or tracking.
