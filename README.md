# MobileDev Cohort 2026

A learning workspace for the Mobile Development Cohort 2026, covering React (web) and React Native (mobile) development.

## Projects

### `react-refresher/`
A React 19 refresher project scaffolded with [Vite](https://vite.dev/).

- **Framework:** React 19
- **Bundler:** Vite 8
- **Purpose:** Refreshing core React concepts before diving into React Native

**Getting started:**
```bash
cd react-refresher
npm install
npm run dev
```

| Script | Description |
|--------|-------------|
| `npm run dev` | Start the development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build |
| `npm run lint` | Run ESLint |

---

### `reactnative-init/`
A React Native project bootstrapped with [Expo](https://expo.dev/) and [Expo Router](https://expo.github.io/router/docs/) for file-based routing.

- **Framework:** React Native + Expo 55
- **Routing:** Expo Router (file-based)
- **Purpose:** Mobile app development for iOS and Android

**Getting started:**
```bash
cd reactnative-init
npm install
npx expo start
```

| Script | Description |
|--------|-------------|
| `npm start` | Start the Expo development server |
| `npm run android` | Open on Android emulator |
| `npm run ios` | Open on iOS simulator |
| `npm run web` | Open in web browser |
| `npm run lint` | Run Expo linter |

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [Expo Go](https://expo.dev/go) app on your physical device (optional)
- Android Studio or Xcode for emulator/simulator support
